# my frontend project initializer
it's a template frontend project that maintains some basic libraries, tests, configurations(eslint). 

## commands
* install dependecies: `npm i` 
* build: `npm run build`
* run on local: `npm run dev`, then open http://localhost:5173/
* run test: `npm run test`
* run test coverage: `npm run coverage`

## basic contents
* react
* daisyui
* lucide
* tailwindcss
* vite
* eslint
